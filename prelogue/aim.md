# 指南的初衷
- 第一，或许还是所谓的开源精神，有一些信息没必要当作秘密甚至被赋予商业价值后进行牟利，开源能够营造并推广一种良好的风气，这在当下是难能可贵的
- 第二，做这么一件事情对于笔者来说，是很有**自我实现**感的。本身帮助人就是一件很酷的事情，maybe在一些人的眼中这是嘉豪行为，但如果能帮助到你，那是我的荣幸~
- 第三，其实是有一些宏大叙事的**情怀**所在的。管院在树洞中永远会有一些stereotype刻板印象：一群精致的利己主义者。
  - 但我想说这并不是正确的，我在过去一年的生活中，遇到了很多很好的同学、学长学姐、同性朋友和异性朋友、老师（*即使不得不承认存在一些神人，但是哦拜托在哪儿都会有的*）。不客气地说，哪怕最好的朋友之间都会存在着所谓的隐瞒，但是这丝毫不妨碍友好欢乐的氛围的存在。
  - 这一份指南，其实没有什么牺牲，无非是一些休闲时间罢了。但是这至少能代表着，还是会存在一些愿意分享的人。

- 小朋友们（其实我也是小朋友），管院从来不缺乏卷怪，内卷永远存在。有些时候可能会有病态的发展倾向，让你感到非常的迷茫；但有的时候，这种内卷环境下的push感，是会把你不断推出舒适圈，push你去选择一个未知的方向并且探索的。换句话来说，商赛、创赛、奖学金、社会实践、实习……等等“内卷”的方向，其实也意味着你完全有多元的发展机会。  
我希望上面的这段话不被理解成一种pua，这只是我纯粹在当下时间节点的个人感受。如果你在内卷的过程中感受到不适和崩溃，请立即停止并且告诉自己，不是自己不够努力，而是本身这个内卷环境和评价体系就存在弊病。“**自由而无用**”这一民间校训虽然近年来不被推崇，但是仍旧是笔者最重要的择校依据。这句话启示着，除了卷xx之外，你完全可以走你自己的道路，做你自己喜欢的事情，自由而无需考虑功利价值。
- 很可惜的是，在这个优绩主义盛行、经济动荡、学历贬值、保研maybe重要的时代，内卷是一条你**不得不去尝试涉足并且痛苦地坚持**的一条道路。
